# Quicker 客制手机端 UI

本仓库实现了自定义的手机端UI, 用于代替物理触发键盘, 本项目为教学目的, 你可以自行阅读本项目源代码, 修改以生成自己的页面

## 使用方式

- 本项目源码目前使用自然语言(即中文)实现
- 请使用任何你喜欢的“编译器”完成成品的生成, 可使用网页端 ChatGPT 等类型产品, 也可以用如 CodeX 等编程客户端直接让其生成代码
- 在生成前, 请简单阅读本文最后的源代码, 首次运行需要你修改下面 2 个部分
    
    1. 填好 Quicker 账号 和 推送码 ([获取方式](https://getquicker.net/Member/PushTools)) - 目的: 触发自己账号的电脑的动作
    2. 改好 设置好要触发的动作的动作ID (获取方式, 自己电脑面板右键动作 点菜单 信息 -> 复制 -> 动作ID) 然后写死在源代码里 - 目的: 生成的作品直接有对应动作, 点击就可以触发

    成功生成后, index.html 文件可以直接在电脑上打开, 电脑上测试成功后, 用自己喜欢的方法在手机上打开这个网页文件即可, 随后可以自己修改源代码, 添加更多的动作.
